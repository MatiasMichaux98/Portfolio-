Proyectos

Selección de algunos de mis proyectos de práctica y personales, desarrollados con C# y el ecosistema .NET.

Sistema de Reservas – API Backend

API REST desarrollada en ASP.NET Core (.NET 8) para la gestión de reservas de peliculas, salas, horarios y asientos, aplicando principios de Clean Architecture y buenas prácticas de diseño para sistemas backend escalables y mantenibles.

  • ✔ Clean Architecture (Application, Domain, Infrastructure, API)
  • ✔ ASP.NET Core sobre .NET 8
  • ✔ Autenticación y autorización con JWT
  • ✔ Refresh Token persistido y enviado mediante cookies HTTP-only
  • ✔ ASP.NET Identity para gestión de usuarios y roles
  • ✔ Control de acceso basado en roles (RBAC)
  • ✔ Entity Framework Core con SQL Server
  • ✔ Patrón Repository y Services
  • ✔ Manejo global de excepciones mediante middleware
  • ✔ Background Services para expiración automática de reservas
  • ✔ Seed inicial de usuarios y roles
  • ✔ Documentación interactiva con Swagger (Bearer JWT)
  • ✔ Unit Tests implementados (en proceso de integración)
  • ✔ Docker y Docker Compose para entorno de desarrollo (API + SQL Server)

Tecnologías: ASP.NET Core (.NET 8) · Entity Framework Core · SQL Server · JWT · ASP.NET Identity · Clean Architecture · Swagger

E-commerce Backend – API REST

Backend REST desarrollado en ASP.NET Core (.NET 8) para una plataforma de comercio electrónico. Provee funcionalidades completas de gestión de usuarios, productos, categorías y carrito de compras, con autenticación y autorización segura mediante JWT y ASP.NET Identity.

El sistema está diseñado con una arquitectura desacoplada basada en servicios, utilizando Entity Framework Core para la persistencia de datos en SQL Server. Incluye integración con Mercado Pago para la gestión de pagos, manejo global de excepciones mediante middleware y documentación interactiva con Swagger.

  • ✔ ASP.NET Core sobre .NET 8
  • ✔ API REST para E-commerce (usuarios, productos, categorías y carrito)
  • ✔ Autenticación y autorización con JWT
  • ✔ ASP.NET Identity para gestión de usuarios y roles
  • ✔ Control de acceso basado en roles (RBAC)
  • ✔ Entity Framework Core con SQL Server
  • ✔ Arquitectura basada en Services y separación de responsabilidades
  • ✔ AutoMapper para mapeo de DTOs
  • ✔ Middleware para manejo global de excepciones
  • ✔ Seed inicial de usuarios y roles
  • ✔ Integración con Mercado Pago para procesamiento de pagos
  • ✔ Documentación interactiva con Swagger (JWT Bearer)
  • ✔ Docker y Docker Compose para entorno de desarrollo (API + SQL Server)
Swagger: /swagger
